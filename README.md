1. На странице "Погода" названия городов можно вводить как латиницей (например, Nizhny Novgorod), так и кириллицей.
2. На странице "Фильмы" названия фильмов необходимо вводить только на английском языке (оригинальные, без локализаций: например, "Die Hard", а не "Крепкий орешек").
3. Если при введении данных и нажатии на кнопку на любой из трех страниц загрузка слишком затянулась (более 10 секунд), нужно обновить эту страницу и повторить сделанное ранее.
